# Tourism Recovery Dashboard 

## Project Overview

This project is a Tableau dashboard that shows how tourism has recovered over time after disruptions. It helps understand trends in tourist arrivals and revenue.

## Tools Used

* Tableau
* Excel / CSV

## Dataset

Tourism dataset with details like country, year, tourist arrivals, and revenue.

## Dashboard Features

* Tourism trends over time
* Country/region comparison
* Revenue analysis
* Interactive filters

## Key Insights

* Tourism is gradually recovering over time
* Some regions recovered faster than others
